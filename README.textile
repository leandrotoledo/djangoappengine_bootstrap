h1. djangoappengine_bootstrap

Script to build a complete but simple test project running on
"djangoappengine":app / "django-nonrel":nonrel with all deps bundled within.

A little advice is use Python 2.5 for Google App Engine, so, if you
use Ubuntu 11.04, that script suggests an installation from a PPA.


h2. Using

<pre>
<code>
    1. Clone
    git clone git@github.com:leandrotoledo/djangoappengine_bootstrap.git
    cd djangoappengine_bootstrap

    2. Run
    chmod u+x djangoappengine_bootstrap.sh
    ./djangoappengine_bootstrap.sh

    3. Setup
    You will be asked to inform the destination of your project.
</code>
</pre>
